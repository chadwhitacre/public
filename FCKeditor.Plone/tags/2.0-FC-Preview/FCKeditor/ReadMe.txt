FCK Editor 2.0 FC (Preview) - Zope CMF product installation
------------------------------------------------------------------------------------

 Plone Product adaptation :
   Jean-mat Grimaldi (jean-mat@macadames.com) - http://macadames.com


 To use FCK Editor with Plone, put the files into your Products directory of your
 Zope Instance and restart the Zope server. For Plone2 installation use
 Plone ControlPanel of your Plone Site, or use Portal_QuickInstaller inside the ZMI.

 Not tested with other CMF portal - we need feedback.

 Note : 
 before installing FCK Editor 2 remove previous versions with Portal_QuickInstaller,
 and remove previous fckeditor directory from your instance/products directory.

 The cohabitation with other wysiwyg editors (Epoz, CMFVE ...) is not provided, to 
 make it operational you must change the wysiwyg_support templates. But this feature
 is not necessary because FCKEditor 2 is MSIE5.5+ and Gecko compatible.

 For more information about using fck editor 2 with Plone, go to http://macadames.com

 Many thanks to Frederico ;-).
